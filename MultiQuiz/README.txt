Step 1: When using this application, make sure to format the "questions.txt" file correctly, follow step 2.
Step 2: To format file correctly by the Aiken standard, make sure to end questions with ?, start options with ),
        and keep your answer as 1 character.
Example:
Hi, What is 1+1?
1) 4
2) 2
3) 5
4) 6
2
Step 3: Make sure to always have 4 options, no more or no less. To add more questions, make sure to add a question below the answer.
Step 4: When on the menu screen, make sure to enter and submit your window's username.
